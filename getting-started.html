Getting Started

Never used a terminal before? You're in the right place.

This guide walks you through finding and canceling subscriptions you forgot about. No coding experience needed—just follow the steps.

What You'll Need
  • Claude Pro subscription — $20/month at claude.ai (flat fee, no extra charges)
  • A Mac or Windows computer
  • Google Chrome browser — for automated cancellations
  • 15-30 minutes — depends on how many subscriptions you have

1 Install Claude Code 5-15 min

Claude Code lets Claude (the AI) work with files on your computer and control your browser to cancel things.

  1. Open Terminal
    Press Cmd + Space, type Terminal, press Enter.
  2. Install Node.js (if you don't have it)
    Go to nodejs.org and click the big green LTS button on the left. Run the installer.
    Important: After installing Node.js, close Terminal and open it again.
  3. Install Claude Code
    Copy this command, paste into Terminal, press Enter: 
    sudo npm install -g @anthropic-ai/claude-code
    Note: You'll be asked for your computer password. This is normal—type it and press Enter. (You won't see the characters as you type.)
  4. Verify it worked 
    claude --version
    You should see a version number like 1.0.x.
  1. Open PowerShell
    Press the Windows key, type PowerShell, click it.
  2. Install Node.js (if you don't have it)
    Go to nodejs.org and click the big green LTS button on the left. Run the installer.
    Important: After installing Node.js, close PowerShell and open it again.
  3. Install Claude Code
    Copy this command, paste into PowerShell, press Enter: 
    npm install -g @anthropic-ai/claude-code
  4. Verify it worked 
    claude --version
    You should see a version number like 1.0.x.

3 Get Your Bank Transactions 3-5 min

Export a CSV file with your recent transactions. Most banks have this option.

Apple Card
  1. Open Wallet app on your iPhone
  2. Tap Apple Card
  3. Scroll down and tap Card Balance
  4. Tap the ... menu → Export Transactions
  5. Choose Last 12 Months and CSV
  6. Send it to yourself (AirDrop, email, or save to iCloud)
Chase
  1. Log in at chase.com
  2. Click your account
  3. Find Download account activity (download icon)
  4. Select date range—last 12 months works best
  5. Choose CSV format
Other Banks

Look for "Export," "Download," or "Download transactions" in your account settings. Choose CSV format (not PDF or Excel).

If you can't find it, try searching "[your bank name] export transactions CSV".

Move your CSV file(s) into the project folder:
Drag the CSV from your Downloads folder into the just-fucking-cancel-main folder on your Desktop.

  • Claude finds recurring charges (Netflix, Spotify, that gym membership...)

Troubleshooting

"claude: command not found"
  • Close Terminal/PowerShell and reopen it
  • Check if Node.js is installed: run node --version
  • Try reinstalling: sudo npm install -g @anthropic-ai/claude-code (Mac) or npm install -g @anthropic-ai/claude-code (Windows)
npm install fails with permission error (Mac)

Make sure you're using sudo at the start of the command:

sudo npm install -g @anthropic-ai/claude-code

Enter your computer password when prompted.

Claude can't find my CSV
  • Make sure the CSV is inside the just-fucking-cancel-main folder
  • Make sure it's a .csv file (not .xlsx or .pdf)
  • Try telling Claude the exact filename: "Look for transactions.csv"
Browser automation not working
  • Make sure Google Chrome is installed (Safari/Firefox won't work)
  • Try closing all Chrome windows first, then let Claude open it
  • Run /mcp and check that browser tools are enabled
Claude asks me to log in but I already did

Just tell Claude "I'm already logged in" or "continue." Sometimes Claude needs a nudge to proceed.

The audit is taking forever

Large CSV files (multiple years of transactions) take longer. If it's been more than 10 minutes with no progress, press Ctrl + C and try with a smaller date range.
